# wdio-which

Before you begin, ensure you have node installed locally, see https://nodejs.org/en/download/ for the latest version

### Install instructions
-Start by cloning the repo to you local machine, using the `git clone` command, appended with the address of this repository(found using the green button in the top right of this page) 

-Run `npm install` in the base directory of the repo, to install all required modules.

-To run the tests, type `npx run test`.
